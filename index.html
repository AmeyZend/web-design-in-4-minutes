Web Design in 4 Minutes

by Jeremy Thomas

Let's say you have a product, a portfolio, or just an idea you want to share with everyone on your own website. Before you publish it on the internet, you want to make it look attractive, professional, or at least decent to look at.

What is the first thing you need to work on?

Content

The purpose of design is to enhance the presentation of the content it's applied to. It might sound obvious, but content being the primary element of a website, it should not be established as an afterthought.

Written content, like the paragraph you're currently reading, makes up for more than 90% of the Web. Styling this textual content will go a long way.

Let's assume you've already finalized the content you want to publish and just created an empty style.css file. What is the first rule you can write?

Centering

Long lines of text can be hard to parse, and thus hard to read. Setting a limit of characters per line greatly enhances the readability and appeal of a wall of text.

body {
  margin: 0 auto;
  max-width: 50em;
}

After styling the text blocks, what about styling the text itself?

Font Family

Using a clean and legible font will make your text more enjoyable to read.

body {
  font-family: Arial, sans-serif;
}

After setting up the font, let's work on colors.

Colors

Choosing a color scheme that complements your content enhances readability and appeal. For simplicity, we can start with a minimal color palette.

body {
  color: #333;
  background-color: #f5f5f5;
}

Now that we have basic colors, let's look at spacing.

Spacing

Adding some padding and margins helps separate elements on the page, making it look organized and readable.

p {
  margin-bottom: 1em;
}

Now let's look at incorporating images.

Graphics and icons can be used either as ornaments to support the content or as stand-alone pieces of content.

To use images effectively, let's add some borders.

Borders

Adding borders can help highlight certain sections or images.

img {
  border: 1px solid #ddd;
  padding: 4px;
}

Finally, let's finish with some conclusion.

Conclusion

With just a few lines of CSS, we've made a big improvement to the look and feel of this web page. Remember that design is an iterative process, and small adjustments can make a huge difference.

Thanks for following along, and happy coding!
